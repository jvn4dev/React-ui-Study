## 요구사항

1. 버튼을 누르면 왼쪽 아래 코너에서 스낵바가 옆으로 나옵니다.
2. 나타날 때, 사라질 때 모두 애니메이션이 적용됩니다.(❌ 미구현)
3. 일정시간이 지나면, 사라집니다.
4. 여러번 눌렀을 때, 스택처럼 위로 스낵바가 쌓입니다.
5. 3개가 초과하여 보여질 때는, 맨 아래 스낵바가 사라지고, 새로 나타납니다.
   - 사라지고 나서 나타납니다. (❌ 미구현)
